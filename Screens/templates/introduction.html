Objective of this toolkit

This toolkit aims to provide banks and financial institutions with an understanding as to where they currently stand in terms of preparedness to complete the KYC & onboarding of metaverse customers.

How should this toolkit be used?

Step 1: User should start first with the "Information Sheet" and enter the required information in the appropriate fields.
Step 2: User should specify the portfolio that needs assessment in the "Asessment Portfolio" dropdown.
Step 3: User should then shift to "Initial screening" sheet and answer all the questions there to determine whether they should proceed with this toolkit.
Step 4: If the "Initial screening" sheet concludes that user should proceed with this toolkit, the user should shift to "Assessment Parameters" tab.
Step 5: There are 4 themes in this toolkit, each representing one focus area on which the stakeholder has to evaluate themselves.
Step 6: The "Scoring guide" column in each sheet provides user with guidance on how to come up with the correct response based on their self-evaluation.
Step 7: User should enter their response in the "Score" sheet. For ease of usage, each cell of the "Score" column is locked with the set of all possible correct responses.
Step 8: Based on the scores assigned for each question, an overall score for the specific theme is indicated at the bottom of the sheet.
Step 9: Repeat the same set of actions for remaining 3 themes.
Step 10: After answering all the questions and assigning the scores, user has to move to the "Final score" sheet.
Step 11: The total scores of each of the 4 themes will automatically be fetched and a final score will be calculated.
Step 12: User can check their preparedness level based on the final score they get.
